# 🦊 Raposa Saltadora (Project Pygame Zero)

> Um jogo de plataforma 2D minimalista e divertido desenvolvido em Python utilizando a biblioteca **Pygame Zero**.

---

## 🚀 Sobre o Jogo

**Raposa Saltadora** coloca o jogador no controle de uma raposa que precisa navegar por plataformas, desviar de espinhos perigosos e derrotar inimigos patrulheiros para alcançar a bandeira vermelha da vitória. O jogo traz mecânicas clássicas dos jogos de plataforma retrô com uma lógica de física e colisão simplificada.

## 🛠️ Tecnologias Utilizadas

* **Linguagem:** [Python](https://www.python.org/) (v3.10 ou superior)
* **Framework:** [Pygame Zero (pgzero)](https://pygame-zero.readthedocs.io/)

## ✨ Mecânicas e Funcionalidades

* **Movimentação Fluida:** Suporte para setas do teclado ou teclas `A` e `D`.
* **Pulo Duplo:** Pule mais alto usando `Espaço`, `Seta para Cima` ou `W` (mesmo estando no ar).
* **Combate Retrô:** Elimine os inimigos pulando diretamente no topo de suas cabeças. É impulsionado para cima ao acertá-los!
* **Obstáculos:** Espinhos terrestres e zonas azuis perigosas geram morte instantânea e reset do jogador.
* **Sistema de Áudio:** Música de fundo em loop contínuo e efeitos sonoros para pulos e dano.
* **Controles Rápidos:** Pressione `M` para ligar/desligar a música e `R` para reiniciar a faixa de áudio.

## 🎮 Controles

| Comando | Ação |
| :--- | :--- |
| `A` / `D` ou `Seta Esquerda` / `Seta Direita` | Mover para a Esquerda / Direita |
| `W` / `Espaço` / `Seta Cima` | Pular / Pulo Duplo |
| `M` | Mutar / Desmutar Música de Fundo |
| `R` | Forçar Reinício da Música |
| `Clique do Mouse` / `Qualquer Tecla` | Iniciar o Jogo (Menu) ou Reiniciar (Vitória) |
